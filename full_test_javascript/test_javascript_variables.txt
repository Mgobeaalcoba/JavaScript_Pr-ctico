¿Qué es una variable y para qué sirve?

Una variable es una reserva de espacio en memoria. Sirve para alojar, cuando lo creamos necesario datos en las mismas y poder usarlos en nuestros programas.

¿Cuál es la diferencia entre declarar e inicializar una variable?

Declarar una variable es reservar el espacio en memoria. Inicializar una variable es asignarle un valor a ese espacio en memoria. 

¿Cuál es la diferencia entre sumar números y concatenar strings?

Ambos se realizan con el signo "+" pero en el primer caso se está trabajando con un tipo de dato numerico mientras que en el segundo caso se está realizando con un tipo de dato String. 

¿Cuál operador me permite sumar o concatenar?
El operador para sumar y/o concatenar es "+"
